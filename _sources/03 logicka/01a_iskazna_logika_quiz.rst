Исказна логика - квиз
=====================

.. quizq::

    .. mchoice:: tautologije
        :answer_a: ¬ (p ∨ q) ⇔ ¬ p ∧ ¬ q
        :answer_b: p ⇒ (q ⇒ r) ⇔ p ∧ q ⇒ r
        :answer_c: ¬ (p ∧ q) ⇔ ¬ p ∧ ¬ q
        :answer_d: p ∧ (q ∨ r) ⇔ (p ∧ q) ∨ (p ∧ r)
        :correct: c

        Која од претходних формула није таутологија:

    .. mchoice:: implikacija
        :answer_a: њена претпоставка тачна
        :answer_b: њена претпоставка нетачна
        :answer_c: њен закључак тачан
        :answer_d: њен закључак нетачан
        :correct: b, c

        Импликација је сигурно тачна ако је:
        
    .. mchoice:: negacija_tautologije
        :answer_a: порецива
        :answer_b: незадовољива
        :answer_c: задовољива
        :answer_d: таутологија
        :correct: b

        Исказна формула је таутологија ако и само ако је њена негација:

    .. mchoice:: valuacija
        :answer_a: логичка еквиваленција
        :answer_b: дискункција
        :answer_c: конјункција
        :answer_d: валуација
        :correct: d

        Функција која свакој исказној променљивој додељује истинитосну вредност (тачно, нетачно) назива се:
        
    .. mchoice:: knf
        :answer_a: задовољивост формула у конјуктивној нормалној форми
        :answer_b: задовољивост формула у дисјунктивној нормалној форми
        :answer_c: порецивост формула у дисјунктивној нормалној форми
        :answer_d: таутологичност формула у конјунктивној нормалној форми
        :correct: a
